## Class

``` python
class CocaCola:
    formula = ['caffeine','sugar','water','soda']

# instance
cola = CocaCola()
print(cola.formula)

# inherit
class CocaColaChina(CocaCola):
    formula = ['caffeine','sugar','water','soda']
```
